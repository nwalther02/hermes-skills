- Prior regressions with narrow known fixes.
- PR/workflow blockers and GitHub auth oddities.

## Do Not Use Memory As Truth When

- The user asks about the current file contents.
- A branch, PR number, or deployment state might have changed.
- The task is a review or audit of live code.

In those cases, use memory to route the search, then verify against the current repo.
